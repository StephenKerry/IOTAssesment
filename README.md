#  Stephens Smart Doorbell

### Stephens Smart Doorbell – My IOT Assessment proposal  
For this assessment I would like to make a smart doorbell similar to how the ring 
doorbell works, but also with the use of rasbery pi. It waits for a button pressthen captures an 
image. It will then send the user the image captured via Email
It will consist of 4 IOT Layers, a RED LED GROVE button, which acts as the "doorbell". The processing layer where the rasberry pi sends sata to a 
server or cloud platform. Then a gateway layer that uses MQTT to a cloud service then 
an application layer where it shows thw image via gmail. 
Hardware needed. Rasberry Pi, Rasberry pi camera module. PIR Red LED Gorve button 
Software: Linux/Python. Gmail 
MQtt
